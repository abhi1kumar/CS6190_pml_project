Repo for the CS6190 PML Course Project 

### Cloning the repo
Type the following command in terminal
```bash
git clone git@github.com:abhi1kumar/pml.git
```

### Extra directories
We need to make some extra directories to store data and models
```bash
mkdir data
mkdir model
```
